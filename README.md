# Carrera Académica - 7502420041

Proyecto académico desarrollado en PHP orientado a objetos, aplicando DDD y Arquitectura Hexagonal.

## Tecnologías
- PHP 8+
- MySQL
- Laragon
- Git y GitHub

## Estructura del proyecto
- `public/`: punto de entrada web
- `src/`: dominio, aplicación e infraestructura
- `database/`: script de base de datos

## Funcionalidades
- Registrar carrera académica
- Listar carreras académicas
- Buscar carrera académica por id
- Editar carrera académica
- Eliminar carrera académica

## Base de datos
Ejecutar el script ubicado en:

`database/bd_carrera_academica.sql`
